Уровни доступа в системе
Главный администратор (ADMING)

Полные права: Управление всеми пользователями системы. Может назначать роли ADMIN и USER, удалять пользователей ADMIN и USER. Первый зарегистрированный пользователь автоматически становится ADMING.

Администратор (ADMIN)

Расширенные права: Доступ ко всем функциям лизинговых договоров (просмотр, создание, редактирование, удаление). Может удалять только пользователей с ролью USER. Не может управлять другими ADMIN или ADMING.

Пользователь (USER)

Базовые права: Полный доступ к работе с лизинговыми договорами (просмотр, создание, редактирование, удаление записей). Не имеет прав управления пользователями.

Иерархия управления
Кто кем управляет:
  • ADMING →
    • Назначает роли ADMIN/USER
    • Удаляет ADMIN и USER
  • ADMIN →
    • Управляет лизинговыми договорами
    • Удаляет USER
  • USER →
    • Работает с лизинговыми договорами
    • Не управляет пользователями
